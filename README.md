
# SVDVillerApi  

This is a free API that grabs data from the Stardew Valley wiki's about villagers.

This includes the villager's:
* Name
* URL from their Wiki page
* Birthday (season and day)
* If they are marrieable or not
* Their loved, liked, neutral, disliked, hated gifts
* Their schedules 

The Swagger documentation can be accessed here: https://svdvillagerapi.onrender.com/docs
